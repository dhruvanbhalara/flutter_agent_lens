# Agent Instructions

# Goal
- This is a MCP server written into dart which utilize the flutter devtools services/extension usage to provide a set of tools to debug and analyze flutter applications. The agent is designed to be used in conjunction with the MCP server and provides a set of tools that can be used to analyze and debug flutter applications.

# Rules for the agent
- Never commit bin/test_all_tools.dart file.
- Never refer bin/test_all_tools.dart file in any documentation.
- bin/test_all_tools.dart is a temporary file that is generated by the agent to test the tools. It is not part of the source code and should not be included in any commits or documentation.
- tools_exectuion*.md is a temporary file that is generated by the agent to test the tools. It is not part of the source code and should not be included in any commits or documentation.
- docs/ directory is used for feature documentation and upcoming integration into MCP server. It is not part of the source code and should not be included in any commits or documentation.
